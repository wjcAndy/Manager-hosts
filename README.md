管理员权限：非管理员运行时自动请求管理员权限重启脚本

添加条目：从脚本目录hosts.txt读取条目并添加到系统hosts文件

多种格式支持：hosts.txt支持"IP 域名"或"域名 IP"两种格式，中间允许多个空格

自动去重：添加时自动检查IP+域名完全一致的重复条目

行号删除：根据行号删除单个或多个指定行（逗号分隔）

范围删除：支持范围删除格式如"53-63"删除连续行

IP删除：根据IP地址删除所有相关域名条目

域名模糊删除：根据域名关键词模糊匹配删除（如"test1"匹配"test1.yuque.com"）

清理显示：显示hosts文件时去除空行和注释行，只显示有效条目

自动备份：每次修改前自动备份，保留最近3个备份文件

备份还原：可从备份文件列表中选择任意备份进行还原

DNS刷新：还原后可选刷新DNS缓存使更改立即生效
